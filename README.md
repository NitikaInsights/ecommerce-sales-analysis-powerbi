# E-commerce Sales Analysis Dashboard | POWER BI

## Project overview

This project presents an interactive E-Commerce Sales Analysis Dashboard Developed using Microsoft Power BI.

The Dashboard transforms raw sales data into meaningful insights related to sales performance, profit, products, payment modes, and regional performance.

## Project Objective

The objective of this project is to analyze e-commerce sales data and create an interactive dashboard that helps in understanding business performance and supports data-driven decision-making.

## Tools & Technologies

- Microsoft Power BI
- Power Query
- DAX
- Data Cleaning
- Data Analysis
- Data visualization

## Key Performance Indicators

- Total Sales: 438k

## Dashborad Features

- Monthly sales trend analysis
- Sales analysis by product category
- Sales analysis by sub-category
- Top 10 customer analysis
- Payment mode analysis
- Profit analysis by category
- Profit analysis by city
- Top cities by sales
- Interactive quarter filter
